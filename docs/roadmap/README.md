# Roadmap

Diese Roadmap beschreibt die geplante Entwicklung von HammerJarvis als lokalen AI Engineering Assistant.

## Abgeschlossen

### v0.3.1 - Engineering Assistant Foundation

- ProTool Assistant als erstes Engineering-Modul.
- Read-only CSV-Analyse.
- GUI-Integration.
- Batch analysis.
- Panel preview und OP7 Digital Twin Preview.
- Browser upload support.
- Knowledge improvements.

## Geplant

### v0.4.0 - ProTool Project Explorer

- Projektweite ProTool-Dateiansicht.
- Automatische Erkennung typischer ProTool-CSV-Dateien.
- Projekt-Summary fuer Texte, Panels, Issues und Platzhalter.

### v0.5.0 - WinCC Flexible Assistant

- Read-only Analyse exportierter WinCC-flexible-Daten.
- HMI-Text- und Uebersetzungs-QA.
- Migrationsrelevante Hinweise.

### v0.6.0 - TIA Project Indexer

- Lokale Indexierung exportierter TIA-Projektinformationen.
- Strukturierte Suche nach Bausteinen, Symbolen und HMI-Elementen.

### v0.7.0 - PLC Cross References

- Querverweise fuer PLC-Symbole, Bausteine und Signale.
- Lesende Analyse von Abhaengigkeiten und Verwendung.

### v0.8.0 - HMI Translation Studio

- Terminologiepruefung.
- Placeholder-Vergleich zwischen Sprachversionen.
- Panel-Laengenpruefung und Review-Reports.

### v1.0 - Engineering OS

- Konsolidierte lokale Engineering-Arbeitsoberflaeche.
- Plugin-orientierte Architektur.
- Knowledge Base, Reports, Diagnosen und assistierte Workflows.
